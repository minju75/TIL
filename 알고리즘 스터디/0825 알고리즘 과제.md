### swea 4861 회문

```python
t = int(input())
for tc in range(1, t+1):
    n, m = map(int,input().split())
    a, b = [] , []
    for i in range(n):
        a.append(input())
    for k in range(n):
        se = ''
        for i in range(n):
            se += a[i][k]
        b.append(se)

    result = ''
    for q in range(n):
        for r in range(n-m+1):
            if a[q][r:r+m+1] == a[q][r:r+m+1][::-1]:
                result += a[q][r:r+m+1]
                break
            if b[q][r:r+m+1] == b[q][r:r+m+1][::-1]:
                result += b[q][r:r+m+1]
                break

    print("#%d %s" %(tc, result))
```



### swea 4865 글자수

```python
t = int(input())
for tc in range(1, t+1):
    a = input()
    b = input()
    sub_cnt = 0
    cnt = 0
    for i in a:
        for j in b:
            if i == j:
                sub_cnt += 1
        if sub_cnt > cnt:
            cnt = sub_cnt
        sub_cnt = 0

    print("#%d %d" %(tc, cnt))
```
